**WEB TECHNOLOGIES WORKSHOP**

Full stackdevelopment with Django

We would discuss following topics:-

  1. Basics of internet.
  2. Web App stack.
  3. HTML.
  4. CSS with Bootstrap.
  5. Java script with jQuery.
  6. AJAX.
  7. Django and MySQL.

We choose students who have following traits->

  • Basic knowledge of Python .
  • Enthusiastic.
  • Innovative.
  • Asking questions.
  • Hard working.
Day to day works as follows:

Day1

  • Basics of Internet

    • We shall discuss how the web works and how web applications operate. This will give a good context to entire course.

  • Structure of web page.

    • This will discuss HTML in detail.

  • Web page in STYLE

    • This will discuss CSS in detail along with BootStrap the popular CSS Framework. mini hackathon to build a page and style it.


Day2

  • Java script

    • Adding behaviour to page

  • DOM

    • manipulating page using code.

  • jQuery

    • making your task of DOM manipulation a piece of cake. AJAX using jQuery and JSON and consuming a web service. Creating a small game using HTML, CSS and JavaScript.

    Day ends with another mini hackathon to have people innovate in whatever they want to make.


Day3

  • Django

    • introducing web server and starting a Django app.

  • Databases

    • MySQL working with database in Django.

    Make Instagram clone in Django using whatever we learnt till now.

NOTE :-

Day ends with a full night hackathon asking people in teams to build an end to end app of their choice.
